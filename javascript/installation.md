## افزودن کتابخانه

برای نصب از طریق `npm` :

```bash
npm install chabokpush --save
```
```bash
yarn add chabokpush
```
و یا با استفاده از [CDN](https://unpkg.com/chabokpush@0.1.6/dist/chabokpush.min.js) ، چابک را به پروژه ی خود اضافه کنید.
