Chabok Push فعال کردن
-------------


چابک از طریق CocoaPods در دسترس است، برای نصب آن به سادگی خط زیر را به Podfile خود اضافه کنید:

``` ruby
pod 'ChabokPush'

```

سپس با روش زیر نصب کنید :

``` ruby
pod install

```
